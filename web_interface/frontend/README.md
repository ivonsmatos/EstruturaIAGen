# Frontend do Projeto

Este diretório contém o código para o frontend do projeto, desenvolvido com React.

## Como Executar

1. Navegue até o diretório do frontend:
   ```bash
   cd web_interface/frontend
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm start
   ```
4. Acesse o frontend em seu navegador:
   ```
   http://localhost:3000
   ```
